Smol Engine

3D engine created from scrath with audio module

Me and a colleague created a 3D engine using C++ and different libraries, such as SDL2, OpenGL, GLEW, GML, imGui. It was designed with a workflow similar to Unity’s, providing tools to manage game object behaviour, scene data saving, and intuitive camera navigation. It supports importing 3D models and textures, and includes a fully integrated audio system based on Wwise, using dedicated listener and source components.
